About Me

My name is Maarten Hogeveen. I'm an aspiring web developer.

Ever since my childhood I've had an affinity for technology. And my mother, a developer and designer herself, exposed me to the world of Web Development at a young age. My interests have always reflected this in one way or another. Fixing computers, gaming, video editing and streaming to name a few.

A couple of my defining characteristics:

  • Organized
  • Flexible
  • Dependable
  • Punctual
  • Calm
  • Efficient
  • Reliable

COLMAR ACADEMY

The Colmar Academy project was an excersize in making my own design decisions for a pre-existing wireframe, and creating a responsive static webpage from scratch.

The design and color palette of the webpage should give the image of a modern institution, through a sleek design and popping but understated color scheme.

CONQUER Template Remake

The Conquer Template Remake was an excersize in making a pixel perfect copy of an existing single page website template. The whole project was made with only HTML and CSS.

Recreating the Conquer template without JavaScript was a good excersize in problem solving. The most apparent solution for a problem was not always within the scope of the project. Understanding and acknowledging this was a great way to learn when it was time to focus on the next part of the website.
